# symfony-training

## install [virtualbox](https://www.virtualbox.org/wiki/Downloads)

## install [vagrant](https://www.vagrantup.com/downloads.html)

## clone repo
```sh
git clone git@github.com:wimme002/symfony-training.git
```
## run vagrant
```sh
cd /path/to/repo/symfony-training/
vagrant up
```

## Done!
